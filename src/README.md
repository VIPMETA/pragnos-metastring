# PragnOS MetaSystem Source Modules

This directory will contain the executable and library code that powers the PragnOS ecosystem.

## Module Staging

| Module | Purpose | Status |
|--------|---------|--------|
| `core/` | Kernel-level orchestration utilities and shared primitives. | _pending_ |
| `interfaces/` | API gateways and integration adapters. | _pending_ |
| `analytics/` | Predictive processing, telemetry and feedback loops. | _pending_ |

Each subdirectory should expose a `mod.rs` or equivalent entry-point that documents the runtime contracts it provides.
